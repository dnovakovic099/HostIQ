# Property Management Guide

Learn how to add, manage, and organize your properties in HostIQ.

## Overview

Properties are the foundation of HostIQ. Each property can have multiple units, and each unit can have multiple rooms with custom inspection requirements.

## Adding a Property

### Step 1: Navigate to Properties

1. Open HostIQ app
2. Tap **"Properties"** tab (bottom navigation)
3. You'll see your properties list

### Step 2: Create New Property

1. Tap the **"+"** button (top right or floating button)
2. Fill in property details:

#### Required Information
- **Property Name**: e.g., "Downtown Apartment Complex"
- **Address**: Full street address
- **City**: City name
- **State/Province**: State or province
- **Zip/Postal Code**: Postal code
- **Country**: Country name

#### Optional Information
- **Property Type**: Apartment, House, Condo, etc.
- **Notes**: Special instructions or information
- **PMS Integration**: Connect to Property Management System

### Step 3: Save Property

1. Review all information
2. Tap **"Save"** or **"Create Property"**
3. Property appears in your list

## Property Types

### Manual Properties
- Created directly in HostIQ
- Full control over details
- Custom room templates
- Direct management

### PMS Properties
- Synced from Property Management System
- Automatic updates
- Limited editing (some fields)
- Integration with PMS

## Managing Properties

### Viewing Properties

The properties list shows:
- **Property Name**: Name of the property
- **Address**: Location
- **Unit Count**: Number of units
- **Badge**: PMS indicator (if synced)
- **Performance**: Average score (if available)

### Property Details

Tap any property to see:
- **Overview**: Basic information
- **Units**: List of all units
- **Inspections**: Recent inspections
- **Performance**: Scores and metrics
- **Settings**: Property configuration

### Editing a Property

1. Open property details
2. Tap **"Edit"** button
3. Modify information
4. Tap **"Save"**

**Note:** PMS properties may have limited editing options.

### Deleting a Property

1. Open property details
2. Tap **"Delete"** or swipe to delete
3. Confirm deletion
4. **Warning:** This also deletes all units and inspection history

## Units

### Adding Units

1. Open a property
2. Tap **"Add Unit"** or **"Units"** tab
3. Enter unit information:

#### Unit Information
- **Unit Name/Number**: e.g., "Unit 101" or "Apartment A"
- **Unit Type**: Studio, 1BR, 2BR, etc.
- **Notes**: Special instructions for cleaners

4. Tap **"Save"**

### Managing Units

- **View Units**: See all units in property
- **Edit Unit**: Modify unit details
- **Delete Unit**: Remove unit (removes inspection history)
- **Assign Cleaners**: Assign specific cleaners to units

## Rooms & Room Templates

### Understanding Rooms

Rooms define what needs to be inspected:
- Each unit has multiple rooms
- Each room can have custom requirements
- Rooms guide the inspection process

### Adding Rooms to a Unit

1. Open a unit
2. Tap **"Rooms"** or **"Add Room"**
3. Select room type:

#### Room Types
- Bedroom
- Bathroom
- Kitchen
- Living Room
- Dining Room
- Hallway
- Closet
- Balcony
- Laundry
- Office
- Other

4. Add room name (optional)
5. Add custom requirements (optional)
6. Tap **"Save"**

### Room Templates

Create reusable room templates:

1. Go to **Settings** → **Room Templates**
2. Tap **"Create Template"**
3. Define:
   - Room type
   - Standard requirements
   - Checklist items
4. Save template
5. Apply to units

### Custom Requirements

For each room, you can add:
- **Checklist Items**: Specific things to check
- **Instructions**: Special cleaning requirements
- **Standards**: Quality expectations
- **Notes**: Additional information

**Example:**
- "Bed must be made with hospital corners"
- "All surfaces dust-free"
- "No personal items visible"

## Property Organization

### Best Practices

1. **Naming Convention**
   - Use consistent naming
   - Include location identifiers
   - Make it easy to find

2. **Unit Organization**
   - Number units logically
   - Group similar units
   - Use clear identifiers

3. **Room Setup**
   - Add all rooms that need inspection
   - Define clear requirements
   - Keep templates consistent

4. **Notes & Instructions**
   - Add property-specific notes
   - Include special instructions
   - Document important details

## Property Performance

### Viewing Performance

Each property shows:
- **Average Score**: Overall cleanliness average
- **Recent Inspections**: Latest results
- **Trend**: Improving or declining
- **Issues**: Common problems

### Performance Metrics

- **Cleanliness Score**: Average AI score
- **Guest-Ready Rate**: Percentage passing
- **Issue Frequency**: Common problems
- **Cleaner Performance**: Who performs best

## Search & Filter

### Searching Properties

1. Use search bar at top
2. Type property name or address
3. Results filter automatically

### Filtering Options

- **By Type**: Manual vs PMS
- **By Performance**: High/low scores
- **By Status**: Active/inactive
- **By Location**: City/region

## PMS Integration

### Connecting PMS

1. Go to **Settings** → **PMS Integration**
2. Select your PMS provider
3. Enter credentials
4. Authorize connection
5. Properties sync automatically

### Managing PMS Properties

- **View**: See all synced properties
- **Sync**: Manually refresh data
- **Settings**: Configure sync options
- **Disconnect**: Remove integration

**Note:** Some PMS properties have limited editing.

## Tips & Tricks

### Organizing Large Portfolios

1. **Use Consistent Naming**
   - Makes search easier
   - Improves organization

2. **Group by Location**
   - Create naming convention
   - Use addresses effectively

3. **Leverage Templates**
   - Save time on setup
   - Ensure consistency

4. **Regular Maintenance**
   - Update information
   - Remove old properties
   - Keep data current

### Efficiency Tips

- **Bulk Operations**: Add multiple units at once
- **Copy Settings**: Duplicate room templates
- **Quick Access**: Use search for fast navigation
- **Shortcuts**: Use quick actions from dashboard

## Common Tasks

### Adding Multiple Properties

1. Add properties one at a time
2. Or import from PMS
3. Use consistent naming
4. Set up rooms after adding

### Setting Up New Property

1. Add property details
2. Add all units
3. Create room templates
4. Assign cleaners
5. Test with first inspection

### Updating Property Information

1. Edit property details
2. Update units as needed
3. Modify room requirements
4. Save changes
