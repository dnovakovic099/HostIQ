# Account Types: Owner vs Cleaner

HostIQ supports two main account types, each with different features and capabilities.

## Property Owner Account

Property Owners manage properties, assign cleaners, and review inspection results.

### Features Available to Owners

#### Property Management
- Add and manage multiple properties
- Create and manage units within properties
- Set up room templates and checklists
- Define inspection requirements
- Track property performance scores

#### Team Management
- Invite cleaners via email
- Assign inspections to cleaners
- View cleaner performance metrics
- Manage team members
- Remove cleaners from team

#### Inspection Review
- View all inspection results
- Access AI analysis reports
- Review cleanliness scores
- See detailed issue reports
- Download inspection reports

#### Analytics & Insights
- Property performance dashboard
- Cleaner performance tracking
- Inspection history and trends
- Low-rating alerts
- Usage statistics

#### Billing & Subscriptions
- Manage subscription plans
- View billing history
- Track usage limits
- Upgrade/downgrade plans
- Payment settings

#### Additional Tools
- Generate Airbnb dispute reports
- Inventory management
- Valuable items tracking
- Pricing analytics (if subscribed)
- PMS integration settings

### Owner Dashboard Overview

The Owner dashboard shows:
- **Welcome Section**: Personalized greeting and quick stats
- **Recent Inspections**: Latest inspection results with scores
- **Quick Actions**: Property, Team, Insights shortcuts
- **Usage Indicator**: Subscription usage tracking
- **Low Rating Alerts**: Properties needing attention

## Cleaner Account

Cleaners perform inspections, capture media, and submit results for analysis.

### Features Available to Cleaners

#### Inspection Management
- View assigned inspections
- Start new inspections
- Continue in-progress inspections
- Create manual inspections
- View inspection history

#### Media Capture
- Take photos with camera
- Record videos
- Select from photo library
- Upload multiple media files
- Room-by-room capture

#### Inspection Submission
- Upload media for analysis
- Review before submission
- Submit for AI analysis
- Track submission status
- View analysis results

#### History & Reports
- View past inspections
- Check cleanliness scores
- Review feedback
- See performance metrics
- Access cleaning reports

#### Profile Management
- Edit profile information
- Change password
- Payment settings (if applicable)
- View payment history

### Cleaner Dashboard Overview

The Cleaner dashboard shows:
- **Today's Jobs**: Pending and in-progress inspections
- **Job Cards**: Property name, unit, due date, status
- **Create Button**: Manual inspection creation
- **History Tab**: Past completed inspections

## Account Switching

Currently, each account is tied to a single role. If you need both Owner and Cleaner access:

1. **Option 1**: Create separate accounts
   - One account as Owner
   - Another account as Cleaner
   - Use different email addresses

2. **Option 2**: Contact Support
   - Request dual-role access
   - Support can enable multiple roles on one account

## Choosing Your Account Type

### Choose Owner If:
- You own or manage rental properties
- You need to assign work to cleaners
- You want to review inspection results
- You need analytics and reporting
- You manage multiple properties

### Choose Cleaner If:
- You perform cleaning services
- You complete inspections on-site
- You need to document your work
- You work for property owners/managers
- You want to track your assignments

## Invitation System

### Owners Inviting Cleaners

1. Go to **"Team"** or **"Manage Cleaners"**
2. Tap **"Invite Cleaner"**
3. Enter cleaner's email address
4. Cleaner receives email invitation
5. Cleaner accepts and creates account
6. Cleaner appears in your team list

### Accepting an Invitation

1. Check your email for invitation
2. Tap the invitation link
3. Create your account (if new)
4. Sign in (if existing)
5. You'll see assigned inspections

## Account Settings

Both account types can access:
- Profile editing
- Password changes
- Notification preferences
- Privacy settings
- App preferences
- Logout
