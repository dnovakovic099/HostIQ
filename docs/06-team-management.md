# Team Management Guide

Learn how to invite, manage, and work with cleaners in HostIQ.

## Overview

As a Property Owner, you can build a team of cleaners to perform inspections. This guide covers everything about managing your cleaning team.

## Inviting Cleaners

### Step 1: Navigate to Team Management

1. Open HostIQ app
2. Go to **"Team"** or **"Manage Cleaners"**
   - From dashboard: Tap **"Team"** quick action
   - From tabs: Go to Settings â†’ Team
3. You'll see your current team list

### Step 2: Send Invitation

1. Tap **"Invite Cleaner"** or **"+"** button
2. Enter cleaner's email address
3. (Optional) Add a personal message
4. Tap **"Send Invitation"**

### Step 3: Cleaner Accepts

1. Cleaner receives email invitation
2. They click the invitation link
3. They create account (if new) or sign in
4. They appear in your team list

## Managing Your Team

### Viewing Team Members

The team list shows:
- **Name**: Cleaner's name
- **Email**: Contact email
- **Status**: Active, Pending, Inactive
- **Performance**: Average scores
- **Inspections**: Number completed

### Cleaner Details

Tap any cleaner to see:
- **Profile**: Name, email, photo
- **Performance**: Scores and metrics
- **Assignments**: Current and past
- **Inspections**: History and results
- **Settings**: Permissions and access

## Assigning Inspections

### Method 1: From Property

1. Open a property
2. Select a unit
3. Tap **"Assign Inspection"**
4. Select cleaner from list
5. Set due date/time
6. Add notes (optional)
7. Tap **"Assign"**

### Method 2: From Dashboard

1. Go to dashboard
2. Tap **"Assign Inspection"**
3. Select property and unit
4. Choose cleaner
5. Set details
6. Assign

### Assignment Details

When assigning, you can set:
- **Cleaner**: Who performs inspection
- **Due Date**: When it should be completed
- **Property/Unit**: Location
- **Notes**: Special instructions
- **Priority**: Urgency level

## Cleaner Permissions

### Access Levels

**Full Access:**
- Can see all properties
- Can create inspections
- Can view all inspections

**Limited Access:**
- Assigned properties only
- Assigned inspections only
- Limited history access

### Managing Permissions

1. Open cleaner details
2. Tap **"Permissions"** or **"Settings"**
3. Adjust access levels
4. Save changes

## Managing Assignments

### Viewing Assignments

See all assignments:
- **Pending**: Not yet started
- **In Progress**: Currently being worked on
- **Completed**: Finished inspections
- **Overdue**: Past due date

### Updating Assignments

1. Open assignment details
2. Modify:
   - Due date
   - Cleaner
   - Notes
   - Priority
3. Save changes

### Canceling Assignments

1. Open assignment
2. Tap **"Cancel"**
3. Confirm cancellation
4. Cleaner notified

## Removing Cleaners

### When to Remove

- No longer working together
- Performance issues
- Inactive account
- Request from cleaner

### How to Remove

1. Open cleaner details
2. Tap **"Remove from Team"**
3. Confirm removal
4. Cleaner loses access to your properties

**Note:** Inspection history is preserved.

## Troubleshooting

### Cleaner Can't See Assignment

**Check:**
- Assignment was created
- Cleaner is assigned
- Property/unit access
- App is updated

**Solutions:**
- Verify assignment details
- Check permissions
- Resend invitation if needed
- Contact support

### Performance Issues

**Address:**
- Review inspection results
- Provide specific feedback
- Offer training
- Set improvement goals

### Communication Problems

**Improve:**
- Use clear notes
- Provide examples
- Regular check-ins
- Open dialogue
