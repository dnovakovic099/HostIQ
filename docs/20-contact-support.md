# Contact Support

Need help? We're here for you! Here's how to get in touch with the HostIQ support team.

## Support Channels

### In-App Support

The easiest way to get help:

1. Open HostIQ app
2. Go to **Settings**
3. Tap **"Help & Support"** or **"Contact Us"**
4. Fill out the support form
5. Submit your request

### Email Support

Send an email to our support team:

**Email:** support@hostiq.com

**Include:**
- Your account email
- Issue description
- Steps to reproduce (if applicable)
- Device/OS information
- Screenshots (if helpful)

### Response Times

- **General Inquiries**: 24-48 hours
- **Urgent Issues**: 24 hours
- **Technical Problems**: 24-48 hours
- **Billing Questions**: 48 hours

## Before Contacting Support

### Self-Help Resources

1. **Check Documentation**
   - Browse user guides
   - Search FAQ
   - Review troubleshooting

2. **Common Solutions**
   - Restart the app
   - Check internet connection
   - Verify permissions
   - Update the app

3. **Search FAQ**
   - Many questions answered there
   - Quick solutions available
   - Covers common issues

## What to Include

When contacting support, please provide:

### Account Information
- Email address (don't share password)
- Account type (Owner/Cleaner)
- App version

### Issue Details
- Clear description
- When it started
- Steps to reproduce
- Expected vs actual behavior

### Device Information
- Device model
- OS version (iOS/Android version)
- App version

### Screenshots/Media
- Screenshots of errors
- Photos of issues
- Screen recordings (if helpful)

## Types of Support

### Technical Issues
- App crashes
- Feature not working
- Upload problems
- Login issues

### Account Issues
- Can't create account
- Password reset
- Account access
- Billing questions

### Feature Questions
- How to use features
- Best practices
- Workflow questions
- Feature requests

### Billing Support
- Subscription questions
- Payment issues
- Plan upgrades/downgrades
- Refund requests

## Feature Requests

Have an idea? We'd love to hear it!

**How to Submit:**
1. Use in-app feedback
2. Email support
3. Describe the feature
4. Explain the benefit

**We Consider:**
- User demand

## Bug Reports

Found a bug? Help us fix it!

**Include:**
- What happened
- What you expected
- Steps to reproduce
- Screenshots/videos
- Device/OS info

**Priority Levels:**
- **Critical**: App unusable
- **High**: Major feature broken
- **Medium**: Minor issue
- **Low**: Cosmetic issue

## Privacy & Security

### Security Issues
