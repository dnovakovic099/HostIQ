# Owner Dashboard Guide

The Owner Dashboard is your command center for managing properties, reviewing inspections, and tracking performance.

## Dashboard Overview

When you open HostIQ as a Property Owner, you'll see the main dashboard with several key sections.

## Welcome Section

At the top of your dashboard, you'll see:

- **Personalized Greeting**: "Hello, [Your Name]!"
- **Quick Stats**: Overview of your portfolio
- **Decorative Elements**: Visual design elements

## Quick Actions Card

A floating card below the welcome section provides quick access to:

### Property Button
- **Icon**: Home icon
- **Action**: Navigate to Properties screen
- **Use**: Add new properties or manage existing ones

### Team Button
- **Icon**: People icon
- **Action**: Navigate to Manage Cleaners
- **Use**: Invite cleaners or manage your team

### Issues Button
- **Icon**: Analytics icon
- **Action**: Navigate to Insights screen
- **Use**: View property issues and analytics

### Usage Indicator
- Shows your current subscription usage
- Displays free image limit and usage
- Tap to view subscription details

## Low Rating Alert

If any properties have ratings ≤ 4.7, you'll see an alert card:

- **Warning Icon**: Red warning indicator
- **Count**: Number of low-rated properties
- **Action**: Tap to view details in Insights
- **Purpose**: Quick access to properties needing attention

## Recent Inspections Section

### Section Header
- **Title**: "Recent Inspections"
- **View All Button**: Navigate to all inspections

### Stats Row
Shows quick statistics:
- **TOTAL**: Total number of recent inspections
- **AVG SCORE**: Average cleanliness score
- **COMPLETE**: Number of completed inspections
- **PROCESSING**: Number of inspections being analyzed

### Inspection Cards

Each inspection appears as a card showing:

#### Property Information
- **Property Name**: Name of the property
- **Unit Name**: Specific unit inspected
- **Cleaner Name**: Who performed the inspection
- **Date**: When inspection was completed
- **Photo Count**: Number of photos uploaded

#### Score Display
- **Cleanliness Score**: Numerical score (0-10)
- **Color Coded**: Visual indicator of quality

#### Status Indicators
- **Attention Badge**: Red badge for failed inspections
- **Status**: Complete, Processing, or Failed

#### Actions Available
- **Tap Card**: View full inspection details
- **Airbnb Dispute Report**: Available for failed inspections
- **Delete**: Remove inspection (with confirmation)

### Empty State

If no inspections exist yet:
- **Icon**: Clipboard outline
- **Message**: "No Inspections Yet"
- **Subtext**: Explains that inspections appear after cleaners submit them

## Pull to Refresh

- **Action**: Pull down on the dashboard
- **Effect**: Refreshes all data
- **Indicator**: Shows refresh spinner

## Navigation Tabs

At the bottom of the screen, you'll see tabs:

### Inspections Tab (Current)
- Shows the dashboard
- Displays recent inspections
- Quick access to inspection reports

### Properties Tab
- List of all properties
- Add new properties
- Property details and management

### Insights Tab
- Analytics and performance metrics
- Property and cleaner insights
- Issue tracking

### Settings Tab
- Account settings
- Subscription management
- App preferences

## Understanding Inspection Statuses

### Complete
- **Color**: Green
- **Meaning**: Inspection finished and analyzed
- **Action**: Can view full report

### Processing
- **Color**: Orange
- **Meaning**: AI analysis in progress
- **Action**: Wait for completion

### Failed / Cleaning Failed
- **Color**: Red
- **Meaning**: Inspection didn't meet standards
- **Action**: Review issues, generate dispute report

## Dashboard Features

### Auto-Refresh
- Dashboard refreshes when you navigate to it
- Data updates every 30 seconds when active
- Manual refresh available via pull-to-refresh

### Onboarding Popup
- Appears for new users
- Guides you through first steps
- Can be dismissed and shown again

### Performance Optimization
- Fast loading with prioritized data
- Background fetching for non-critical data
- Smooth scrolling and interactions

## Tips for Using the Dashboard

1. **Check Daily**: Review recent inspections regularly
2. **Use Quick Actions**: Fast access to common tasks
3. **Monitor Alerts**: Pay attention to low-rating warnings
4. **Review Scores**: Track average scores over time
5. **Use Stats**: Quick overview of portfolio health

## Common Actions

### Viewing Inspection Details
1. Tap any inspection card
2. See full report with photos
3. Review AI analysis
4. Check specific issues

### Adding a Property
1. Tap "Property" quick action
2. Or go to Properties tab
3. Tap "+" button
4. Fill in property details

### Managing Team
1. Tap "Team" quick action
2. Or go to Settings → Team
3. Invite new cleaners
4. View team performance

### Viewing Insights
1. Tap "Issues" quick action
2. Or go to Insights tab
3. Review analytics
4. Check property performance
