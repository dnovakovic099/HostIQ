# Cleaner Dashboard Guide

Your dashboard is your home base for viewing assignments, starting inspections, and tracking your work.

## Dashboard Overview

When you open HostIQ as a Cleaner, you'll see your main dashboard with your assigned jobs and inspection options.

## Home Screen Layout

### Today's Jobs Section

The main area shows your current assignments:

#### Job Cards

Each job appears as a card showing:

**Property Information:**
- **Property Name**: Name of the property
- **Unit Name**: Specific unit to inspect
- **Address**: Property location

**Status Badge:**
- **PENDING**: Not yet started (yellow/orange)
- **IN PROGRESS**: Started but not completed (blue)

**Due Date:**
- **Today**: Shows "Today at [time]"
- **Tomorrow**: Shows "Tomorrow at [time]"
- **Other**: Shows date and time

**Progress Info:**
- **Photo Count**: Number of photos uploaded (if in progress)
- **Notes**: Special instructions from owner

**Action Button:**
- **"Start Inspection"**: For pending jobs
- **"Continue Inspection"**: For in-progress jobs

### Empty State

If you have no jobs:
- **Icon**: Calendar outline
- **Message**: "No jobs yet"
- **Hint**: "Tap the + button below to create a new inspection"

## Creating Manual Inspections

### When to Create

Create manual inspections when:
- Owner hasn't assigned yet
- You want to document work
- Quick inspection needed
- Self-initiated inspection

### How to Create

1. **Tap the "+" Button**
   - Floating button (bottom right)
   - Or from home screen

2. **Select Property**
   - Choose from available properties
   - Or any property you have access to

3. **Select Unit**
   - Choose specific unit
   - Verify unit details

4. **Start Inspection**
   - Tap "Create Inspection"
   - Navigate to capture screen
   - Begin taking photos

## Understanding Job Status

### PENDING

- **Meaning**: Assigned but not started
- **Action**: Tap "Start Inspection"
- **Color**: Yellow/Orange badge

### IN PROGRESS

- **Meaning**: Started but not submitted
- **Action**: Tap "Continue Inspection"
- **Color**: Blue badge
- **Info**: Shows photo count

### SUBMITTED

- **Meaning**: Sent for AI analysis
- **Status**: Processing or Complete
- **Location**: History tab

## Job Card Details

### Property & Unit

- **Property Name**: Large, bold text
- **Unit Name**: Below property name
- **Address**: With location icon

### Due Date

- **Format**: "Today at 2:00 PM"
- **Format**: "Tomorrow at 10:00 AM"
- **Format**: "Dec 15 at 3:00 PM"
- **Icon**: Clock icon

### Notes Section

If owner added notes:
- **Background**: Light blue/info color
- **Icon**: Information icon
- **Content**: Special instructions
- **Purpose**: Important information

### Waiting for Rooms

If property doesn't have rooms set up:
- **Message**: "Waiting for owner to add rooms"
- **Background**: Warning color
- **Action**: Contact owner

## Navigation

### Pull to Refresh

- **Action**: Pull down on screen
- **Effect**: Refreshes job list
- **When**: Get latest assignments

### Tab Navigation

Bottom tabs provide access to:

**Home Tab (Current):**
- Your dashboard
- Today's jobs
- Quick actions

**History Tab:**
- Past inspections
- Completed work
- Performance metrics

**Reports Tab:**
- Inspection reports
- Detailed results
- Scores and feedback

**Settings Tab:**
- Account settings
- Profile management
- App preferences

## Quick Actions

### Start Inspection

1. Tap job card
2. Tap "Start Inspection"
3. Begin capture process

### Continue Inspection

1. Tap in-progress job
2. Tap "Continue Inspection"
3. Resume where you left off

### Create Manual

1. Tap "+" button
2. Select property/unit
3. Create new inspection

## Understanding Assignments

### Assigned Inspections

- **Source**: Owner assigns to you
- **Appearance**: Shows in job list
- **Due Date**: Set by owner
- **Notes**: Owner's instructions

### Manual Inspections

- **Source**: You create yourself
- **Appearance**: Also in job list
- **Flexibility**: More control
- **Use Case**: Quick documentation

## Job Organization

### Sorting

Jobs are sorted by:
- **Due Date**: Earliest first
- **Priority**: Urgent first
- **Status**: Pending before in-progress

### Filtering

(If available):
- **By Property**: Filter by property
- **By Status**: Pending, in-progress
- **By Date**: Today, this week

## Tips for Using Dashboard

### Daily Routine

1. **Check Morning**: Review today's jobs
2. **Plan Day**: Organize by location/time
3. **Start Early**: Begin inspections
4. **Complete**: Submit before due

### Efficiency Tips

1. **Group by Location**: Do nearby properties together
2. **Check Notes**: Read owner instructions
3. **Prepare**: Charge phone, check supplies
4. **Document Well**: Take good photos

### Best Practices

1. **Regular Checks**: Check for new assignments
2. **Timely Start**: Don't wait until last minute
3. **Complete Properly**: Follow all steps
4. **Review Feedback**: Learn from scores

## Common Scenarios

### Multiple Jobs Today

- **Prioritize**: By due time
- **Organize**: By location
- **Plan**: Route efficiently
- **Complete**: All on time

### Job Appears Late

- **Check**: Refresh screen
- **Contact**: Owner if needed
- **Start**: As soon as possible
- **Communicate**: If issues

### Can't Start Job

**Check:**
- Property has rooms
- You have access
- App is updated
- Internet connection

**Solutions:**
- Wait for owner to add rooms
- Contact owner
- Check permissions
- Restart app

## Troubleshooting

### Jobs Not Showing

**Check:**
- Refresh screen
- Check assignments
- Verify account
- Contact owner

**Solutions:**
- Pull to refresh
- Wait a moment
- Check email for assignments
- Contact support

### Can't Start Inspection

**Check:**
- Rooms are set up
- Permissions granted
- App updated
- Internet available

**Solutions:**
- Contact owner about rooms
- Check app permissions
- Update app
- Check connection

### App Not Loading

**Solutions:**
- Restart app
- Check internet
- Update app
- Restart device
